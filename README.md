# Robot AI Copilot - Firefighting System

**AI-powered auto-aiming system for firefighting robot**

## Features
- Real-time screen capture (no webcam needed)
- YOLOv8 person detection
- Multi-target tracking
- AI aiming system with recoil compensation
- 14 PUBG weapons supported
- Real-time visualization
- Mouse click to fire

## Weapons (14 types)
**AR:** M416, AK47, Beryl M762, AUG, SCAR-L, Groza, FAMAS
**DMR:** MK12, ACE32, SLR
**Sniper:** Dragunov, Mini 14
